Bienvenue dans TestApp

Cette application multiplateforme est un modèle pour les développeurs Qt/C++.

Installation

  • Windows : lancez l’installateur `.msi`
  • macOS : ouvrez le fichier `.dmg`
  • Linux : utilisez le paquet `.deb` ou `.AppImage`
  • Android : installez l’APK
  • Web : ouvrez la version HTML

Utilisation

Lancez l'application et cliquez sur “Click me” pour tester l'interface de base.
