Guide Utilisateur – TestApp

Bienvenue dans TestApp, une application Qt/C++ multiplateforme. Ce guide explique comment l’installer, l’utiliser, exécuter les tests et générer des installeurs.

Prérequis

  • CMake 3.16 ou plus
  • Qt 6.8.x
  • Compilateur C++20 (GCC, Clang, MSVC...)
  • Git (optionnel)
  • Ninja (recommandé)
  • QDoc (pour la documentation développeur)

Lancement

Exécutez le fichier TestApp. Une fenêtre s’ouvre avec le message "Hello World".

Android

adb install -r TestApp-debug.apk

Tests

Trois exécutables sont disponibles :

  • ./CppTest – tests unitaires C++
  • ./QtTests – tests avec QTest
  • ./QmlTests – tests QML

Compilation manuelle

git clone https://github.com/votre-utilisateur/cyber-ribouldingue-test-beta.git
cd cyber-ribouldingue-test-beta
cmake -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build

Générer un installeur

cd build
cpack

Générer la documentation développeur

qdoc docs/testapp.qdocconf

Structure du projet

src/        Code C++
qml/        Interface QML
tests/      Tests C++ et QML
config/     Fichiers Android, Qt, CMake
docs/       Documentation
.github/    Workflows CI/CD
